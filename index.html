my name is paul, i am the tutor for this class

this site is created purely in html and has none of the special features you'll find on almost any professionally designed website today. the idea behind coding a website like this is that it reveals the inherent aesthetic properties of html itself, much like brutalist architecture celebrates raw materials. a major proponent of this web design philosophy today is the net art pioneer Olia Lialina, whose work you can find in the Arena channel.
